# 🌦️ Raspberry Pi IoT Weather Station

## 📌 Overview
This project is an IoT-based weather monitoring system using Raspberry Pi. It collects real-time environmental data like temperature, humidity, and pressure and displays it on a dashboard.

---

## ⚙️ Hardware Used
- Raspberry Pi (3/4 recommended)
- DHT11 / DHT22 sensor (temperature & humidity)
- Optional: BMP180 / BME280 (pressure sensor)
- Jumper wires
- Breadboard

---

## 💻 Software Used
- Python
- Raspberry Pi OS
- Flask (for web dashboard)
- Sensor libraries

---

## 🚀 Features
- Real-time temperature monitoring
- Humidity tracking
- Web-based live dashboard
- Lightweight IoT system
- Expandable for cloud integration

---

## 🛠️ How It Works
1. Sensor collects environmental data
2. Raspberry Pi reads sensor data using Python
3. Data is processed and displayed on a web dashboard
4. Users can access live data via browser

---

## 🌐 Dashboard Access
http://<raspberry-pi-ip>:5000
